## What changed

Claude Code, Anthropic's command-line coding agent, released version 2.1.218 on July 22, 2026.

**Behavior changes:**
- `/code-review` now runs as a background subagent (a helper agent that works without blocking your session) instead of running inline, and it keeps using stacked slash commands as what it reviews.
- `/deep-research` now only starts when you invoke it yourself — Claude no longer launches it automatically.
- Skills that declare `context: fork` now run in the background by default. Add `background: false` in the skill definition to keep the old behavior of blocking until it finishes.
- Boolean-style settings in frontmatter (the metadata block at the top of a file) now also accept `yes`, `no`, `on`, `off`, `1`, and `0` (case-insensitive), in addition to `true`/`false`.
- `claude mcp list` and `/mcp` now show the HTTP status and error text for MCP (Model Context Protocol) server connections, making connection problems easier to diagnose.
- Agent markdown files with a colon used in a disallowed spot are now rejected instead of silently accepted.
- Sandbox command restrictions were tightened, and the trust dialog now names the specific repository it's asking about.

**Fixes:**
- Windows file paths containing a segment starting with `\u` (for example, `C:\Users\unicorn`) were being corrupted into CJK (Chinese/Japanese/Korean) characters.
- Pressing the left arrow key could discard part of the conversation with no way to undo; it now asks for confirmation first.
- Multi-line pasted text could get collapsed, with the letter `j` replacing newline characters.
- `/context` (which shows token usage) was reporting an incorrect number.
- `/ultrareview` failed when given a descriptive, multi-word argument.
- `/code-review ultra` was silently running local reviews instead of cloud reviews in some non-interactive sessions.
- Deeply nested directory trees being deleted or moved could crash the CLI.
- Remote sessions could keep sending heartbeat signals after their backing worker process had already been replaced.
- A race condition during shutdown, a fork-session history bug, and a bug resuming sessions with a malformed attachment were all fixed.

**Accessibility:**
- Screen readers now announce text deletions when running in `--ax-screen-reader` mode.
- Fixed VoiceOver (Apple's screen reader) echoing extra space characters while typing.
- Fixed plugin and settings panels not moving the cursor correctly for screen reader users.

## Why it matters

The `/code-review` and `/deep-research` changes are the most user-visible: code review no longer ties up your terminal while it runs, and deep research (a more expensive, longer-running mode) won't start on its own anymore, so you won't be surprised by it kicking off unprompted. The Windows path-corruption bug and the undo-less left-arrow deletion are both real risks of losing work, so they're worth updating for even if you don't care about the rest.

## How to use it

- Update with `claude update`, or through your package manager.
- If a skill using `context: fork` now runs in the background and you don't want that, add `background: false` to its definition.
- Run `/deep-research` explicitly when you want it — it won't start on its own anymore.
- No action needed for the other fixes; they apply automatically after updating.

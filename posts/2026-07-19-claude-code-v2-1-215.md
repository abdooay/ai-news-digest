## What changed

Claude Code, Anthropic's command-line coding agent, released version 2.1.215 on July 19, 2026. The only change in this release: Claude no longer runs the `/verify` and `/code-review` skills on its own. You now have to invoke them yourself, by typing `/verify` or `/code-review`.

`/verify` is a built-in check that exercises a code change to confirm it actually works. `/code-review` reviews a diff for bugs and cleanup opportunities. Before this release, Claude could decide to run either of these automatically during a task, without being asked.

## Why it matters

If Claude ran these checks on its own, it could add time and cost to a task the user didn't request. Making them opt-in gives people more predictable control over when these extra steps happen. It's a small change, but it affects anyone who was used to Claude proactively verifying or reviewing its own work — that will no longer happen unless you ask for it.

## How to use it

- Update Claude Code to v2.1.215 or later (`claude update`, or via your package manager).
- To check that a change works, run `/verify`.
- To review a diff for bugs and simplification opportunities, run `/code-review`.
- Nothing else changes — both skills still work exactly as before when invoked manually.
